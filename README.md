# Supermarket Sales Data Analysis 

## Overview
This project performs an **Exploratory Data Analysis (EDA)** on a supermarket sales dataset to extract business insights using Python.

## Skills Demonstrated
* **Data Manipulation:** Using `Pandas` for cleaning and organizing data.
* **Data Visualization:** Creating professional charts with `Seaborn` and `Matplotlib`.
* **Insight Extraction:** Analyzing customer segments and product performance.

## Key Insights
1. **Customer Types:** Comparing shopping habits between Members and Normal customers.
2. **Product Performance:** Identifying the most popular product categories.
3. **Revenue Distribution:** Visualizing the spread of total sales.

## Tech Stack
* **Language:** Python
* **Libraries:** Pandas, Seaborn, Matplotlib
* **Environment:** Google Colab
